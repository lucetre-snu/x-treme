# x-treme
### How to start
* `cd x-treme`
* `npm install`
* `npm start`

### Branches
#### 1. homepage
* `git clone -b homepage https://github.com/lucetre/x-treme.git`
#### 2. collection
* `git clone -b collection https://github.com/lucetre/x-treme.git`
#### 3. diary
* `git clone -b diary https://github.com/lucetre/x-treme.git`

### Dependencies
* react-bootstrap
